PhishHunt

PhishHunt is a phishing analysis and investigation tool under CyberEDT Tools, built using HTML, CSS, and JavaScript to analyze emails, URLs, and domains for malicious indicators such as phishing links, spoofed domains, suspicious redirects, and risky sender behavior. It provides clear, risk-based insights to support security analysis and learning.

Features

Email header and sender domain analysis

URL and link inspection

Domain reputation and basic DNS checks

Detection of common phishing patterns

Risk-based analysis with clear explanations

Lightweight and browser-friendly design

Use Cases

SOC analyst triage and phishing investigation

Cybersecurity education and student projects

Phishing awareness demonstrations

Blue-team learning labs

Tech Stack

Frontend: HTML, CSS, JavaScript

APIs : WHOIS, DNS, threat intelligence services

Installation
git clone https://github.com/yourusername/phishhunt.git
cd phishhunt


Open index.html in a browser or deploy using a static web server.
